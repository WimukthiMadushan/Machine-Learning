# 🧮 Cost Function for Linear Regression

This project demonstrates how the **cost function** is calculated and optimized in a simple **linear regression** model using **gradient descent**. The model predicts values for `y` based on the independent variable `x`, and the cost function is used to measure how well the model is performing by comparing the predicted values to the actual values.

---

## 🧰 **Technologies Used**
- **Programming Language**: Python 🐍
- **Libraries**:
  - `numpy`: For numerical computations and array manipulations
  - `matplotlib`: For visualizing the cost function against the model parameter `m`

---

## 🔥 **Features**
- Compute and visualize the **cost function** of a linear regression model.
- Apply **gradient descent** to optimize the parameters `m` (slope) and `c` (intercept) to minimize the cost.
- Plot the **cost** as a function of the parameter `m` during each iteration of the optimization process.
