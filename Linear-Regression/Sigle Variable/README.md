# 📈 Single Variable Linear Regression Model

A simple implementation of a **Single Variable Linear Regression Model** to predict the number of views a video will receive based on the number of videos uploaded. This project demonstrates the core concepts of linear regression, including data visualization, model training, and prediction.

---

## 🧰 **Technologies Used**
- **Programming Language**: Python 🐍
- **Libraries**: 
  - `NumPy` - For numerical computations
  - `Pandas` - For data manipulation
  - `Matplotlib` & `Seaborn` - For data visualization
  - `Scikit-learn` - For model training and evaluation
