# 📊 Multi-Variable Linear Regression Model (Predicting Views)

This project demonstrates a **Multi-Variable Linear Regression Model** that predicts the number of views on a video based on three input features:
- **Videos**: The number of videos uploaded
- **Days**: The number of days since the video was uploaded
- **Subscribers**: The number of subscribers on the account

Using these features, the model predicts the dependent variable:
- **Views**: The number of views the video is expected to receive.

---

## 🧰 **Technologies Used**
- **Programming Language**: Python 🐍
- **Libraries**: 
  - `pandas`: For data manipulation
  - `numpy`: For numerical computations
  - `matplotlib`: For data visualization
  - `scikit-learn`: For building and training the linear regression model
